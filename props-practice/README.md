# Props Practice

A React learning project focused on understanding and practicing component props.

## Description

This project demonstrates how to use props in React components to pass data from parent to child components.

## Getting Started

### Prerequisites

- Node.js and npm installed
- Basic knowledge of React

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

## Topics Covered

- Props basics
- Passing data between components
- Props validation
- Default props
- Destructuring props
