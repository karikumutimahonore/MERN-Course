# JSX Practice

A learning project for mastering JSX syntax and React fundamentals as part of the MERN course.

## Project Structure

```
jsx-practice/
├── src/
│   ├── components/
│   ├── App.jsx
│   └── index.js
├── package.json
└── README.md
```

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

## Learning Goals

- Understand JSX syntax and transpilation
- Practice component creation
- Learn props and state management
- Explore React hooks
